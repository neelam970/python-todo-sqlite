# todo_app.py

A simple command-line To-Do List app built with Python and SQLite.

## Features
- Add tasks
- View all tasks with timestamps
- Mark tasks as completed
- Delete tasks
- Data saved permanently in a database

## How to Run
1. Make sure you have Python installed
2. Run: `my_todo_app.py`
